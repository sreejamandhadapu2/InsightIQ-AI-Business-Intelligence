# 🚀 InsightIQ – AI Business Intelligence Platform

InsightIQ is an AI-powered Business Intelligence platform that enables users to ask business questions in natural language and receive SQL queries, interactive dashboards, visualizations, AI-generated insights, and downloadable reports.

The project combines Data Engineering, Business Intelligence, SQL, Artificial Intelligence, and Data Analytics into a single application.

---

## ✨ Features

### 🤖 AI Powered Analytics
- Natural Language to SQL using Gemini AI
- AI SQL Validation
- AI Business Insights
- AI Query Interpretation

### 📊 Executive Dashboard
- Interactive KPI Cards
- Business Health Score
- Business Alerts
- Executive Summary
- Revenue Trends
- Revenue by Region
- Revenue by Category
- Top Products
- Dashboard Filters

### 📈 Reporting
- PDF Report Export
- CSV Export
- Interactive Charts

### 🏗 Data Engineering
- SQL Server Data Extraction
- ETL Pipeline
- Star Schema Data Warehouse
- PostgreSQL Data Warehouse
